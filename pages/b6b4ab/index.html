MarkDown公式的调优

# 渲染数学公式

我发现主题自带的vuepress-plugin-mathjax插件在渲染/begin会出现问题。

因此使用 markdown-it-katex (opens new window) 解决，插件名@neilsustc/markdown-it-katex.

安装方法：

yarn add -D @neilsustc/markdown-it-katex
1

config.js的部分配置如下：

  markdown: {
    lineNumbers: true, // 代码行号
    extendMarkdown: md => {
            md.set({
                html: true
            })
            md.use(require('@neilsustc/markdown-it-katex'), {"throwOnError" : false, "errorColor" : " #cc0000"})
        }
  },
  head: [
        ['link', {
            rel: 'stylesheet',
            href: 'https://cdn.jsdelivr.net/npm/katex@0.10.0-alpha/dist/katex.min.css'
        }],
        ['link', {
            rel: "stylesheet",
            href: "https://cdn.jsdelivr.net/github-markdown-css/2.2.1/github-markdown.css"
        }]
    ],
