MMDetection的安装

# MMDetection安装

提示

以下所有配置均在Ubuntu-20.04系统下进行。

# 安装Anaconda

从清华镜像源下载安装包：Anaconda3-2021.05-Linux-x86_64.sh (opens new window)

终端执行：

bash Anaconda3-2021.05-Linux-x86_64.sh
1

一直按Enter直至要求输入yes or no。

凡是要求输入yes or no，全部输入yes。

重启终端，若用户名前面出现base，即代表安装成功。

# 配置环境

# 创建conda环境

conda create -n open-mmlab python=3.7 -y
1

# 切换conda环境

conda activate open-mmlab
1

# 安装Pytorch和torchvision

pip3 install torch==1.9.1+cu111 torchvision==0.10.1+cu111 torchaudio==0.9.1 -f https://download.pytorch.org/whl/torch_stable.html
1

提示

我使用conda安装会出现下载进度一直不动的情况，因此这里我使用了pip进行安装。

# 安装MMDetection

pip install openmim
mim install mmdet
1
2

mim 能够自动地安装 OpenMMLab 的项目以及对应的依赖包。

当然，也可以手动安装，详情见官方文档：安装MMDetection (opens new window)

# 验证安装

from mmdet.apis import init_detector, inference_detector, show_result_pyplot

config_file = 'configs/cascade_rcnn/cascade_mask_rcnn_x101_64x4d_fpn_20e_coco.py'
# 从 model zoo 下载 checkpoint 并放在 `checkpoints/` 文件下
# 网址为: http://download.openmmlab.com/mmdetection/v2.0/faster_rcnn/faster_rcnn_r50_fpn_1x_coco/faster_rcnn_r50_fpn_1x_coco_20200130-047c8118.pth
checkpoint_file = 'checkpoints/cascade_mask_rcnn_x101_64x4d_fpn_20e_coco_20200512_161033-bdb5126a.pth'
device = 'cuda:0'
# 初始化检测器
model = init_detector(config_file, checkpoint_file, device=device)
# 推理演示图像
# inference_detector(model, 'demo/demo.jpg')

show_result_pyplot(model, 'demo/test2.jpg', inference_detector(model, 'demo/test2.jpg'))
